Purpose / Goal
Ensure all design work captures Lloyds’ updated brand identity: fresh, modern, digital-first, recognisably Lloyds, grounded in British culture, evocative of trust, movement, and authenticity.

Design Guidelines

These are rules or recommendations to keep designs consistent across contexts.

Logo usage:
‒ Use updated “Lloyds” logo / mark where appropriate; in some formal / legal usage “Lloyds Bank” may still appear. 
Lloyds Bank
+1

‒ Respect logo space (clearance), proportions, direction. No distortions.

Colour usage / hierarchy:
‒ Primary brand green is dominant in key brand touchpoints (headers, large CTAs, etc.).
‒ Accent colours used sparingly to support hierarchy: call-to-actions, highlights, buttons, UI elements, links.
‒ Backgrounds should allow good contrast for text or overlays.

Typography hierarchy:
‒ Headings (H1, H2, etc.) should use heavier weights or expressive versions of the custom typeface to create impact.
‒ Body text should be very legible, with appropriate line spacing.
‒ Subheadings / captions: use lighter weights or smaller scale while maintaining consistency.

Imagery style:
‒ Photos: natural lighting, real people, relatable environments. Avoid overly posed, overly “perfect” stock shots.
‒ Overlays / image treatments: if text is over images, ensure readability (contrast, shadow, overlay).
‒ Illustration style: simple, clean, expressive, matching colour palette; used to clarify or support messaging rather than distract.

Motion / Interaction:
‒ Subtle animations / transitions inspired by horse movement motifs. Could be hover states, loading transitions, micro-interactions.
‒ Motion should feel smooth, considered, not flashy for its own sake.

Consistency:
‒ All elements (colour, typography, imagery, icons, motion) must align.
‒ Create component library / style guide to ensure reuse of buttons, cards, UI controls, etc.

Accessibility:
‒ Colour contrast sufficient for readability.
‒ Fonts readable at smaller sizes.
‒ Responsive behaviour: designs should scale / adapt across devices.
