# The Art of Planning - Session 2 Presentation

An interactive web-based presentation for "The Art of Planning: How I Stopped Fighting AI and Started Shipping"

## Features

- 20 animated slides with smooth transitions
- Keyboard navigation (← → Space Enter)
- Touch/swipe support for mobile
- Progress bar and slide counter
- Lloyds Banking Group brand colors
- Fully responsive design
- No dependencies - pure HTML/CSS/JS

## Deploy to Vercel

### Option 1: One-Click Deploy
1. Push this folder to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Click "New Project"
4. Import your GitHub repository
5. Set the root directory to `presentation`
6. Click "Deploy"

### Option 2: Vercel CLI
```bash
cd presentation
npx vercel
```

### Option 3: Drag & Drop
1. Go to [vercel.com](https://vercel.com)
2. Drag the `presentation` folder to the deployment area

## Local Preview

```bash
cd presentation
npx serve .
```

Then open http://localhost:3000

## Navigation

| Key | Action |
|-----|--------|
| → or Space | Next slide |
| ← | Previous slide |
| Home | First slide |
| End | Last slide |

On mobile: Swipe left/right
